Modal

A dialog-based modal using the native <dialog> element. Provides built-in focus trapping, Esc key to close, and a backdrop -- all handled by the browser. No dependency, no lock-in. Just copy paste.

Support: <dialog> is supported in all major browsers.

Size Variants

Add .vui-modal--sm (380px) or .vui-modal--lg (680px) to adjust the maximum width.

API

ClassDescription
.vui-modalThe <dialog> container
.vui-modal-headerHeader row with title and close
.vui-modal-titleModal heading text
.vui-modal-closeClose button (uses CSS pseudo-element)
.vui-modal-bodyScrollable content area
.vui-modal-footerAction buttons row
.vui-modal--smSmall width variant
.vui-modal--lgLarge width variant

Notes

The .showModal() and .close() calls are the only moving parts. These are native DOM methods on the <dialog> element, not a library API. No dependency, no lock-in. Just copy paste.

Focus is automatically trapped inside the modal while open. Pressing Escape closes the modal. The backdrop prevents interaction with the content behind it.
