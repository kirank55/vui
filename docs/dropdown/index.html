Dropdown

A non-modal action menu using the Popover API. Open and close behavior is declarative today; trigger-relative positioning becomes more precise in newer browsers with anchor positioning support.

Preview

Implementation Notes

The component relies on popover, popovertarget, and popovertargetaction for native open and close behavior. That keeps the menu HTML-only and avoids custom event handling.

Chrome 120-era browsers can open the menu declaratively, but do not yet expose the newer implicit anchor behavior that places the menu flush to its trigger. This component therefore includes a centered fallback and a precise anchor-based layout when supported.
