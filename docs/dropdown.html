Dropdown

A popover-based dropdown menu using the native Popover API. Click-outside-to-close is handled by the browser. No dependency, no lock-in. Just copy paste.

Support: Popover API is supported in all major browsers in current releases.

With Links

Dashboard Settings Profile
HTML 
<button class="vui-dropdown-trigger" popovertarget="nav-dropdown">
  Navigate
</button>

<div class="vui-dropdown-menu" id="nav-dropdown" popover>
  <a href="/dashboard" class="vui-dropdown-item">Dashboard</a>
  <a href="/settings" class="vui-dropdown-item">Settings</a>
  <a href="/profile" class="vui-dropdown-item">Profile</a>
</div>

API

ClassDescription
.vui-dropdown-triggerButton that opens the dropdown
.vui-dropdown-menuThe popover panel (needs popover attribute)
.vui-dropdown-itemMenu item (button or link)
.vui-dropdown-item--dangerDestructive action variant
.vui-dropdown-dividerHorizontal separator
.vui-dropdown-labelGroup heading text

Notes

The popovertarget attribute on the trigger and the popover attribute on the menu are the only connection needed. No dependency, no lock-in. Just copy paste.

Clicking outside the menu or pressing Escape will close it automatically. The browser handles the open/close toggle, backdrop dismissal, and accessibility announcements.
