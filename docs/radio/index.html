Radio Buttons

A focused radio-group component using native inputs inside a fieldset. Selection state stays native while the surrounding labels provide the visual shell.

Preview

Implementation Notes

The radios preserve native grouping, keyboard behavior, and form submission. Styling is applied around the control instead of reimplementing it.
