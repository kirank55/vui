Input Group

A field composition pattern for prepend text, append text, and appended buttons. The visual shell is shared, but the input remains a native text field.

Preview

Implementation Notes

Each addon is just text or a native button placed beside the input in a flex row. The component keeps one shared border so the group reads as a single control.
