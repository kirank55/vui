Checkbox

A checkbox group that keeps the platform behavior intact while styling each choice as a readable surface with larger hit targets.

Preview

Implementation Notes

This pattern is intentionally minimal: native checkboxes, stacked labels, and no JavaScript. Each row is large enough for touch input without custom widgets.
