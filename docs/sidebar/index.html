Sidebar

A standalone sidebar component derived from the docs web component's visual structure, but shipped as plain HTML and CSS for copy-paste use.

Implementation Notes

This component captures the same visual language as the docs sidebar without the custom element. The active state is just a class, so consumers can control it from any router or static page.
