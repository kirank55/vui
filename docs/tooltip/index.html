Tooltip

A lightweight help bubble built with the Popover API. Popover is now Baseline 2025 across major browsers, while anchor positioning remains a progressive enhancement for more exact placement.

Implementation Notes

The strict version of a tooltip people expect is hover and focus driven, but the HTML-only mechanism for that is the newer interest invoker API, which is not available in this project's support range. This component therefore uses click to reveal the tooltip while staying declarative.

The preview includes a primary button trigger, an image trigger where the image sits inside a wrapping span, and an underline-only trigger, all using the same declarative popover approach.
