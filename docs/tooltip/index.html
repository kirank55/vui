Tooltip

A lightweight help bubble built with a popover trigger and upgraded with CSS anchor positioning where the browser makes that possible.

Preview

Implementation Notes

The strict version of a tooltip people expect is hover and focus driven, but the HTML-only mechanism for that is the newer interest invoker API, which is not available in this project's support range. This component therefore uses click to reveal the tooltip while staying declarative.

CSS anchor positioning also lands after the current baseline. The component includes an anchor-positioned bubble when supported and a centered fallback when it is not.
