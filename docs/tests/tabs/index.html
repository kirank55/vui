Tabs — Web Component

Vanilla <vui-tabs> custom element. Shadow DOM, keyboard nav, ARIA. Zero dependencies.

3-tab example

Overview

Use <vui-tabs> when you need a tabs component that works in React, Angular, Vue, or plain HTML. Just drop in the script tag and write HTML.

Specs

Shadow DOM encapsulation. Slot-based content projection. Any number of tabs — the pill indicator width is computed dynamically.

Notes

Keyboard navigation: ← / → to move between tabs, Home / End for first and last. Fires a vui-tab-change CustomEvent on switch.

5-tab example

Introduction

A tab set with five items. The pill indicator width adjusts automatically to 100% / 5 = 20%.

Usage

Add <script type="module" src="vui-tabs.js"></script>, then write <vui-tabs> with slot="tab" buttons and slot="panel" divs.

API

Attributes: label — sets the aria-label on the tablist (defaults to "Tabs").

Events

Fires vui-tab-change (bubbles) with detail.index whenever the active tab changes.

Accessibility

Full ARIA tab pattern: role="tablist", role="tab", role="tabpanel", aria-selected, aria-controls, aria-labelledby.
