Advanced SVG Demonstrations

Zero-dependency, complex visual effects utilizing native SVG filters directly inside the DOM.

1. Gooey Blend Effect

Uses <feGaussianBlur> and <feColorMatrix>. Apply the filter to the parent stringently forces the two child circles to merge natively like liquid when their blurred edges overlap.

2. Dynamic Duotone Filtering

Uses <feColorMatrix> and <feComponentTransfer> to map dark-to-light luminosity values directly to a Navy and Orange palette in real-time. Hover the image to remove the filter.

3. Procedural Fractal Noise

Uses <feTurbulence> to generate a native, resolution-independent noise texture directly in the browser (no external image assets downloaded). Applied to a pseudo-element with mix-blend-mode.
