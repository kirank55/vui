Modal

A dialog surface built on the native <dialog> element. The snippet opens with a small inline onclick handler, stays centered in the viewport, and closes when the backdrop is clicked.

Preview

Using It Today

This component ships as a styled <dialog> surface, including native close handling through method="dialog", a small inline opener that calls showModal(), and a backdrop click that requests the native cancel flow.

Centering is handled in CSS with margin: auto and inset: 0 on the dialog, so the component stays visually balanced without any extra wrapper logic.

The dialog itself remains fully native. The opener and backdrop handling both use direct HTML event handlers so the snippet stays portable when pasted into tools like CodePen without requiring a separate script block.
