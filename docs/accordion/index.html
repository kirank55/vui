Accordion

A disclosure component using the native <details> and <summary> elements.

Support: <details> and <summary> are broadly supported. details[name] is newer.

Notes

The <details> element handles open and close state natively. No dependency, no lock-in. Just copy paste.

Add the name attribute with the same value to multiple <details> elements to create an exclusive accordion where only one item can be open at a time.
