Platform Notes

Modal and accordion rely on mature platform elements. Current major browsers support <dialog>, <details>, and <summary> well.

Dropdown uses the Popover API, which is now broadly available in current releases of the major browsers. Form Controls use :user-invalid, which is also broadly available in current releases.

Tooltip is the exception: Popover is broadly available, but CSS Anchor Positioning is newer and still ahead in Chromium-based browsers. If you need the widest compatibility, start with the other components first.
